[My blog](https://faizilham.github.io) based on [minima](https://github.com/jekyll/minima)

### Commands
1. Run & watch in local
    ```
    bundle exec jekyll serve --drafts
    ```

2. Manual build
    ```
    rake
    ```

3. Manual deployment
    ```
    rake deploy
    ```
